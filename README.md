## 功能介绍
1. 自动获取用户头像和昵称
2. 发布新消息
3. 查看“我的消息”，“我的发布”，“我的收藏”
4. 根据消息类别进行查询
5. 根据关键字搜索消息
6. 允许删除发布的消息
7. 新消息的提醒
8. 匿名发布
9. 管理员删除发布的消息，限制用户发布（数据库后台操作设置权限）
